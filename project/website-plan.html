Car Wash Services

Website Project Subject:

Car Wash Services

  • This website will help people find and book car wash services.

Brief Description of the Planned Site Content:

  • Home Page: A preshow of the website.
  • Services Page: A list of car wash services with prices and descriptions.
  • Contact Page: A form where users can request a service or ask questions.

Reason for Choosing This Subject:

My young brother is interested in starting a car wash business. I want to help him out by creating a website for him and also improve my web development skills while building something useful.

Color Schema:

The color schema for this site includes two main colors:

  • Primary Color (hsla(210, 100%, 30%, 1)): Used for headings and accents, giving the site a professional and clean look.
  • Secondary Color (hsla(0, 0%, 98%, 1)): Used as the background color to provide a clean and pleasant visual experience.

Typography:

1. Primary Font: Oswald (sans-serif)

  • Usage: Oswald will be used for all headings across the site to give a bold, modern look and help guide users as they explore different sections.
  • Applied to:
    • Headings (h1, h2, h3, h4): All headings on the website will use Oswald to clearly define sections and make it easy for users to navigate.

2. Secondary Font: Source Sans Pro (sans-serif)

  • Usage: Source Sans Pro will be used for all body text and paragraphs, ensuring that the content is easy to read and visually appealing for users.
  • Applied to:
    • Body Text (p): All paragraphs, descriptions of services, and general content on the site will use Source Sans Pro for readability.
    • Links and Buttons: Interactive elements like links and buttons will also use Source Sans Pro to maintain consistency in design.
